> 基于 `hyperlane` 框架开发的全栈在线群聊项目
> （[点击此处在线体验](https://chat.ltpp.vip/)）

### 克隆项目

```sh
git clone git@github.com:eastspire/hyperlane-quick-start.git;
```

### 进入项目目录

```sh
cd ./hyperlane-quick-start;
```

### 切换分支

```sh
git checkout group-chat;
```

### 运行

```sh
cargo run
```

### 浏览器访问

```sh
http://127.0.0.1:60007/
```
