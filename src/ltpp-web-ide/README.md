## 功能

LTPP 在线 WEB 编辑器支持以下功能

- 多种编程语言（C、C++、JS、TS、Go、Rust、PHP、JAVA、Ruby、Python3、C#、Ruby）
- 根据系统主题自动切换主题
- 支持编译错误和运行错误等详情信息的显示
- 支持输出显示

## 使用

- 标准输入快捷键：F1-F12 中（除去 F1，F5）
- 运行快捷键：F5
