> 一个提供 HTTP 基本类型的库，包括请求体、响应头和其他核心 HTTP 抽象。

## 安装

要使用此 crate，可以运行以下命令：

```shell
cargo add http-type
```

## 使用示例

```rust
use http_type::*;
```
