> 一个基于 Rust 的库，用于构建和发送 HTTP 响应，提供流畅的接口。该库实现了一个 HTTP Response 类型，支持灵活配置 HTTP 版本、状态码、头部和主体。它支持构建 HTTP 响应负载并通过 `TcpStream` 发送。
