> 一个提供常见 HTTP 常量的全面库，包括头部名称、版本、MIME 类型和协议标识符。

## 安装

要使用此 crate，可以运行以下命令：

```shell
cargo add http-constant
```

## 使用示例

```rust
use http_constant::*;
```
