## 目录结构

```txt
src                  # 项目代码目录
├── app              # app目录
│   ├── controller   # 控制器
│   ├── middleware   # 中间件
├── config           # 配置目录
│   ├── host         # host 配置
│   ├── port         # 端口配置
│   ├── log          # 日志配置
│   ├── plugin       # 插件配置
│   ├── middleware   # 中间件注册配置
│   ├── route        # 路由注册配置
│   ├── server       # 服务创建配置
├── plugin           # 插件目录
static               # 静态资源目录
└── img              # 图片静态资源
```
