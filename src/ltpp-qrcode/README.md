## 功能

- 支持文件解析和保存
- 支持历史记录单个和全部导入导出
- 支持域名前缀自动添加
- 支持 `URL` 点击打开页面
- 二维码自动隐藏（聚焦输入框或者鼠标进入二维码所在区域即可显示），防止设备误扫码
- 一键返回顶部
